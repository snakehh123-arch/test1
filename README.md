# Wanderlust - Travel Around the World

## v1.1 - Local Assets Update

A modern, minimalist travel website showcasing 12 amazing destinations around the world.

### 📁 Project Structure

```
wanderlust/
├── index.html          # Main HTML file
├── styles.css          # Styling (modern & responsive)
├── script.js           # Interactive JavaScript functionality
├── assets/
│   └── images/         # Local image directory
│       ├── paris.svg
│       ├── tokyo.svg
│       ├── newyork.svg
│       ├── dubai.svg
```

### ✨ Features

- **12 Popular Destinations** with detailed information
- **Search & Filter** - Find destinations in real-time
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Interactive Modal** - View detailed destination information
- **Modern UI** - Clean minimalist design with purple gradient theme
- **Local Assets** - All images are stored locally (SVG format for fast loading)
- **Smooth Animations** - Hover effects and transitions
- **Contact Form** - Get in touch section
